# Tripadvisor Recommendation System

This project builds a recommendation system for Tripadvisor-style travel data using Python and Jupyter Notebook.

## Project Execution

1. Create and activate a virtual environment.
2. Install the required packages:

```bash
pip install pandas nltk matplotlib seaborn scikit-learn jupyter
```

3. Open the notebook:

```bash
jupyter notebook notebooks/recommendation_system.ipynb
```

4. Run the notebook cells from top to bottom.

The notebook loads the datasets from the `data/` folder and downloads the required NLTK resources during execution.
